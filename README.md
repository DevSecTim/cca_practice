# CCA-F Practice

Practice code and exercises for the Certified Claude Associate - Foundations (CCA-F) exam.

## Study Plan

The master study plan for this repo is the **[CCA-F 14-Day Study Plan](https://www.finstor.in/wp-content/uploads/2026/03/CCA-F_14Day_Study_Plan.pdf)**.

All exercises and code in this repo follow the structure and topics laid out in that plan.
